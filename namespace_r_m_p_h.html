Crea un framebuffer de profundidad para el sistema de sombras.

Inicializa un framebuffer (depthMapFBO) y una textura (depthMap) que se utilizarán para almacenar el mapa de profundidad necesario para la generación de sombras.

Parameters
depthMapFBOReferencia al Framebuffer Object (FBO) que contendrá el mapa de profundidad.

SHADOW_WIDTHAncho de la textura de sombras (resolución del mapa de sombras).
SHADOW_HEIGHTAlto de la textura de sombras (resolución del mapa de sombras).

Sistema de renderizado sin sombras.

Este sistema se encarga de renderizar las entidades que contienen componentes de renderizado y transformación, sin tener en cuenta las sombras. Se utiliza para escenas donde no es necesario procesar la proyección de sombras.

Parameters
renderconpVector de componentes de renderizado (RenderComponent).
posinfoVector de componentes de transformación (TranformComponent).
shaderShader a utilizar para el renderizado.
lightinfoVector de componentes de luz (LightComponent).
camComponente de cámara activo.
camTrTransformación asociada a la cámara.
widthScreenAncho de la pantalla/viewport.
heightScreenAlto de la pantalla/viewport.

Sistema de renderizado con generación de sombras.

Este sistema se encarga de renderizar la escena incluyendo las sombras proyectadas por las luces. Utiliza un framebuffer para almacenar el mapa de profundidad y aplicar el efecto de sombra en la escena final.

Parameters
renderconpVector de componentes de renderizado (RenderComponent).
posinfoVector de componentes de transformación (TranformComponent).
shaderShader principal para renderizado con sombras.
lightinfoVector de componentes de luz (LightComponent).
camComponente de cámara activo.
camTrTransformación asociada a la cámara.
depthMapFBOFramebuffer Object para almacenar el mapa de profundidad (sombras).
depthMapTextura del mapa de profundidad.
shaderDepthShader para generación del mapa de sombras.
widthScreenAncho de la pantalla/viewport.
heightScreenAlto de la pantalla/viewport.
